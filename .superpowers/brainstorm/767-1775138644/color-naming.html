How should colors be named?

Each extracted color needs a human-readable name — where does it come from?

A

Nearest CSS / Named Color

Match each extracted color to the closest entry in a built-in color name list (e.g., "Crimson", "Steel Blue", "Goldenrod"). Runs fully offline, instant, no API needed.

B

The Color API (online lookup)

Call a free public API like thecolorapi.com to get curated, poetic color names (e.g., "Dusty Rose", "Ocean Mist"). Requires internet. Names are richer but adds a network dependency.

C

Both — API with offline fallback

Try the Color API first for richer names. If offline or the request fails, fall back to the local nearest-color lookup automatically.
